# mrp_fma_replan_custom

Module tampon minimal pour permettre à une base clonée qui référence encore
`mrp_fma_replan_custom` de charger le registre.

Ce module ne contient ni modèles, ni vues, ni données.
Il sert uniquement à éviter l'erreur :
`Some modules are not loaded ... ['mrp_fma_replan_custom']`
